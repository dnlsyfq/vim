### Navigating in Command 

```
H  1-LEFT
K  1-UP
J  1-DOWN
L  1-RIGHT

B  FIRST LETTER
E  END LETTER
W  SKIP nth LETTER

$ JUMP-PARAGRAPH

```

### Manipulating Text

```
* In Command Mode

r  # Edit
a  # append
u  # undo
d  # delete
x  # delete 1 character 
```
